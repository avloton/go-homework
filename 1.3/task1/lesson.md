# Git Lesson

## Git commands
    git init - инициализация репозитория
    git status - информация о текущем состоянии репозитория
    git add -добавить файлы в репозиторий
    git commit - зафиксировать текущее состояние
    git log - вывести историю фиксаций (коммитов)
    git checkout - изменить текущую версию (перемещение между коммитами)

## Tables

**Таблица популярных команд Git:**

| Команда | Использование | Описание |
|---------|---------------|----------|
| `git clone` | `git clone <url>` | клонировать удаленный репозиторий |
| `git branch` | `git branch <name>` | создать новую ветку |
| `git merge` | `git merge <branch>` | объединить ветки |
| `git remote` | `git remote add <name> <url>` | добавить удаленный репозиторий |
| `git fetch` | `git fetch` | получить изменения без слияния |
